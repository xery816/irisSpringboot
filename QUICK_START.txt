快速开始指南
============

首次运行（3步）
--------------

1. 复制依赖
   双击运行: copy-all-dependencies.bat
   
2. 检查完整性
   双击运行: CHECK_DEPENDENCIES.bat
   
3. 启动项目
   双击运行: build-and-run.bat


后续运行（1步）
--------------
直接运行: build-and-run.bat


项目已完全独立
-------------
✓ 不再依赖外部SDK目录
✓ 所有库文件在 native-libs/
✓ 配置文件在 src/main/resources/
✓ 工作目录在 temp/


测试项目
-------
1. 启动后访问: http://localhost:8080
2. 打开: test-client.html
3. 点击"初始化设备"
4. 开始使用注册和识别功能


目录结构
-------
iris_springboot/
├── native-libs/          <- 所有依赖库
│   ├── windows-x64/      <- Windows DLL + 音频
│   └── linux-x64/        <- Linux .so
├── temp/                 <- 运行时数据
├── src/                  <- 源代码
├── target/               <- 编译输出
└── test-client.html      <- 测试客户端


常见问题
-------

Q: 提示找不到DLL？
A: 运行 copy-all-dependencies.bat

Q: 初始化失败？
A: 检查虹膜设备是否连接，运行 CHECK_DEPENDENCIES.bat

Q: 端口被占用？
A: 修改 application.yml 中的 server.port

Q: 如何部署到服务器？
A: 复制整个项目目录即可
